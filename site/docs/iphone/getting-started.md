---
title: iPhone アプリをインストールする
description: インストール直後に見える画面と、最初に確認することをまとめます。
---

CodexPocket の iPhone アプリを入れて起動します。最初はまだ Mac を取り込んでいないため、空の状態から始まります。ここで何も出ていなくても正常です。

## 起動直後の見え方

- Mac をまだ追加していない案内が中央に出ます。
- `QR から追加` と `Host を開く` から Host 追加へ進めます。
- `プロジェクト` はまだ空です。
- `設定` では表示や接続設定を先に変えられます。

初回の空の状態は、設定ミスではなく、これから Mac を取り込む段階だと考えると分かりやすくなります。

## インストール直後に確認すること

- 空の案内と `QR から追加` ボタンが見える
- `Host を開く` から Host 一覧を開ける
- `設定` で言語、テーマ、フォントサイズを変えられる

## まだ Mac を追加していなくてもできること

- 言語を切り替える
- テーマと文字サイズを調整する
- 通知や未読バッジの扱いを確認する

ただし、実際の作業を始めるには Mac 側の準備と Host の追加が必要です。

## 次に進む前に

すでに Mac で Codex の使い方に慣れているなら、iPhone 側で新しく設定するのは接続先だけです。repository や Codex CLI を iPhone に入れ直す必要はありません。

次は [初回起動で確認すること](./first-launch) を開いて、権限と見やすさだけ整えてください。
