# iPhone から始める

CodexPocket の iPhone app は、Mac 上の Codex 実行環境を持ち歩くための「操作面」です。Host を追加し、Project を選び、既存 thread の確認や新規 thread の開始、Composer からの follow-up 送信を iPhone 側で行います。

## iPhone 側でまず把握すること

- `Host`
  どの Mac へつなぐかを表す単位です
- `Project`
  thread、branch、skills、commands の起点になる workspace 単位です
- `Thread`
  既存の transcript を読み返したり、新規 turn を送ったりする主画面です
- `Composer`
  text turn、slash command、skills、exec の入口です
