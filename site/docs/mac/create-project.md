CodexPocket では、iPhone に表示される単位を Project と呼びます。Mac の Codex App で使っている workspace は、そのまま Project の候補になります。

![Codex App の workspace が CodexPocketMac と iPhone に見えてくる流れ](/img/docs/workspace-sync.svg)

## いちばん簡単なやり方

1. Mac で普段使うフォルダを Codex App の workspace に入れる
2. `CodexPocketMac` の `プロジェクト` を開く
3. 同じフォルダが Project として見えていることを確認する

Project 名は自動で入ることがあります。あとから分かりやすい名前に変えたいときは、iPhone 側の追加や編集を使います。

## 最初は 1 件で十分

Pairing 前に 1 件見えていれば、そのまま iPhone で Project 一覧を取り込めます。よく使うものだけ先に入れて、残りはあとから増やして構いません。

## どんなフォルダが向いているか

- いつも thread を続けるリポジトリ
- 進行中の作業を外出先でも確認したいフォルダ
- 調査やレビューで参照することが多い workspace

## 自動で見えないとき

- `CodexPocketMac` を開いたまま数秒待つ
- `プロジェクト` タブを開き直す
- それでも出ないときは [Project を追加する](./register-project)
