Codex App の workspace が自動で入ってくる人は、このページを読む必要はありません。手動で追加するのは、次のようなときだけです。

- Codex App を使わず、`codex` CLI を中心に作業している
- workspace が自動で見えてこない
- iPhone では別の名前で表示したい

## 追加前に Mac で確認すること

- `CodexPocketMac` が起動している
- `Bridge` が `実行中` になっている
- iPhone 側で対象の Host を追加済み

## 追加方法

1. iPhone の `プロジェクト` を開く
2. 右上の `+` から `プロジェクトを追加` を押す
3. Host を選ぶ
4. `作業ディレクトリ` を指定する
5. 必要なら `プロジェクト名` を入れる
6. `保存` を押す

`Browse on Mac` を使うと、Mac 側のフォルダを見ながら選べます。`プロジェクト名` は空でも保存でき、その場合はフォルダ名をもとに自動で決まります。

## どんなときに使うか

- よく使うフォルダだけ先に見せたい
- 自動反映される名前では分かりにくい
- Codex App の一覧とは別に、手元で整理したい

## 追加したあと

保存した Project は、そのまま iPhone の一覧に反映されます。見えないときは、一覧を引いて更新し、接続先の Host が合っているかも確認してください。
