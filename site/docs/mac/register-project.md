Codex App の workspace が自動で入ってくる人は、このページを読む必要はありません。`プロジェクト` タブから手で追加するのは、次のようなときだけです。

- Codex App を使わず、`codex` CLI を中心に作業している
- workspace が自動で見えてこない
- iPhone では別の名前で表示したい

## 追加方法

1. `プロジェクトを追加` を押す
2. `作業ディレクトリ` を指定する
3. 必要なら `プロジェクト名` を入れる
4. `保存` を押す

`参照…` を使うと Finder からフォルダを選べます。`プロジェクト名` は空でも保存でき、その場合はフォルダ名をもとに自動で決まります。

## どんなときに使うか

- よく使うフォルダだけ先に見せたい
- 自動反映される名前では分かりにくい
- Codex App の一覧とは別に、手元で整理したい

## 追加したあと

Bridge がすでに動いているなら、追加した Project はそのまま iPhone から取得できます。iPhone 側で一覧を更新しても見えないときは、接続先の Host が合っているかも確認してください。
