---
title: 対応環境
description: 現在のリリースで前提にしている環境をまとめます。
---

現在の CodexPocket は、Mac の Codex を iPhone から開く構成を対象にしています。初回設定は同じローカルネットワークで行い、設定後は Managed Relay によって別 Wi-Fi やモバイル回線からも接続できます。ここでは、使える組み合わせと、まだ対象外の組み合わせを整理します。

## 対応している組み合わせ

| 項目 | 対応状況 | 補足 |
| --- | --- | --- |
| Mac | 対応 | `macOS 26` 以降 |
| iPhone | 対応 | `iOS 26` 以降の実機 |
| Mac と iPhone が同じローカルネットワーク | 対応 | 初回設定とローカル接続で利用 |
| Managed Relay 経由の別 Wi-Fi / モバイル回線 | 対応 | Host に Relay 情報が保存済みなら自動で切り替え |
| Mac 側で Codex が動く環境 | 対応 | CodexPocketMac は Mac 上の Codex を利用 |
| 日本語 / English 表示 | 対応 | アプリ内で切り替え可能 |

## 一部だけ対応しているもの

| 項目 | 状態 | 補足 |
| --- | --- | --- |
| `git` が入っていない Mac | 一部対応 | 基本操作は進められる場合がありますが、ブランチ操作などの Git 連携は使えません |
| QR を使わないペアリング | 一部対応 | カメラが使えなくても、ペアリングコードの貼り付けで追加できます |

## 現在は対象外のもの

| 項目 | 対応状況 | 理由 |
| --- | --- | --- |
| iPad | 対象外 | 現在の iOS アプリは iPhone 向けです |
| Android | 対象外 | 現在は iPhone 専用です |
| Mac だけで使うクライアント | 対象外 | CodexPocket は iPhone から操作する前提です |
| Managed Relay なしのインターネット越し接続 | 対象外 | 直接公開した Bridge URL への接続は対象外です |
| Managed Relay 未設定のモバイル回線接続 | 対象外 | 先に同じローカルネットワークで Host を設定する必要があります |

## 迷ったときの見方

次の条件を満たしていれば、現在の対象内と考えて問題ありません。

- Mac が `macOS 26` 以降
- iPhone が `iOS 26` 以降
- Mac で Codex を普段どおり使えている
- 初回設定時は Mac と iPhone が同じローカルネットワークにいる
- 別 Wi-Fi やモバイル回線で使う Host には Managed Relay 情報が保存されている

逆に、Managed Relay がない状態で外出先から別ネットワーク越しにつなぎたい場合や、iPad で使いたい場合は、現時点では対象外です。
