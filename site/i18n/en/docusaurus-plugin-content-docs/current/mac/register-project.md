---
title: Register a Project
description: Use this page when you want to add a project from the iPhone instead of relying on Codex App reflection.
---

If Codex App workspaces already appear automatically, you do not need this page. Manual add is only for cases such as:

- you do not use Codex App and work mainly with the `codex` CLI
- the workspace does not appear automatically
- you want a clearer display name on the iPhone

## What to confirm on the Mac first

- `CodexPocketMac` is running
- `Bridge` is `Running`
- the target `Host` is already added on the iPhone

## How to add it

1. Open `Projects` on the iPhone.
2. Tap `+` in the upper-right, then choose `Add Project`.
3. Choose the Host.
4. Set the `Working Directory`.
5. Add a `Project Name` if you want one.
6. Tap `Save`.

If you use `Browse on Mac`, you can choose while looking at folders on the Mac. `Project Name` can be left empty, and in that case the folder name is used automatically.

## When manual add is useful

- you want to show only a small set of folders first
- the automatically imported name is hard to understand
- you want to organize the iPhone list differently from the Codex App list

## After adding it

The saved `Project` appears in the iPhone list right away. If it does not, pull to refresh and also confirm that you are looking at the correct Host.
