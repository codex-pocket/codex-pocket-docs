title: Finish First Launch
description: This page explains which permissions and settings are worth checking on first launch.
---

On first launch, it is usually enough to check the minimum permissions and settings that keep the flow smooth later.

## Common permissions

- camera
  used when adding the Mac with a QR code
- local network
  used when finding and connecting to the Mac on the same network

If you plan to paste a pairing code instead of using QR, the camera can be allowed later.

## Settings worth adjusting early

You can change these in `Settings`:

- language
- theme
- font size
- default follow-up behavior
- `KeepAlive`
- `Timeout`

If you often read long logs on the Mac, adjusting font size first is usually enough.

## Things that are normal at this stage

- zero Hosts
- zero Projects
- no Threads yet

From here, use `Add from QR` from the empty screen or open `Hosts` from the sidebar and import the Mac.
