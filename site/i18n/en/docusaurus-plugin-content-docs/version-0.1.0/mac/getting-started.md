To use CodexPocket from the iPhone, `CodexPocketMac` needs to be running on the Mac.

## First things to do

1. install `CodexPocketMac`
2. launch the app
3. confirm that `General / Bridge / Projects / Pairing / Logs / Details` all open

If the `CodexPocket Mac` window opens and you can already see `Runtime` in `General`, you have reached the right starting point.

## Things worth checking right away

- `Enable Bridge`
- `Launch at login`
- warnings

If you see messages such as `No projects found` or `Bridge prerequisites are not satisfied`, do not keep going yet. Check the next pages first.

## What the menu bar can do

You can also operate `CodexPocketMac` from the menu bar:

- `Start Bridge`
- `Stop Bridge`
- `Show Pairing QR`
- `Open Settings`
- `Open Logs`

That makes it easy to pop up the pairing QR quickly, even when the main window is closed.
