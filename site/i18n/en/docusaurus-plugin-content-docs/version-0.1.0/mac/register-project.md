title: Register a Project
description: Use this page when you want to add a project from the iPhone instead of relying on Codex App reflection.
---

A `Project` is the unit of workspace selection on the iPhone. Only add one manually when automatic reflection from Codex App is not enough.

## What to confirm on the Mac first

- `CodexPocketMac` is running
- `Bridge` is `Running`
- the target `Host` is already added on the iPhone

## How to add it

1. open `Projects` on the iPhone
2. tap `+` in the upper-right, then choose `Add Project`
3. choose the Host
4. enter a `Project Name`
5. set the `Working Directory`
6. tap `Save`

`Browse on Mac` helps you choose while looking at folders on the Mac.

## Which projects you should add first

The best first candidates are workspaces you already use for Codex on the Mac:

- repositories you open every day
- directories you inspect often for review or research
- workspaces whose status you want to check while away from the desk

## If you use Codex App

Workspaces already registered in the Mac Codex App can sometimes be added automatically to the `Projects` list in `CodexPocketMac`. Add them manually from the iPhone only when they do not appear.

## One project is enough at first

If you already have one `Project` before pairing, the iPhone can show the project list right after the connection finishes. You can always add more later.
