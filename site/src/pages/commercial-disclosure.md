本ページは、CodexPocket の通信販売に関する表示事項です。

| 項目 | 内容 |
| --- | --- |
| 法人名・販売事業者名 | CodexPocket |
| 事業者の氏名 | 請求があった場合には、法令に基づき遅滞なく開示いたします。 |
| 所在地 | 請求があった場合には、法令に基づき遅滞なく開示いたします。 |
| 電話番号 | 請求があった場合には、法令に基づき遅滞なく開示いたします。 |
| メールアドレス | [codexpocket@gmail.com](mailto:codexpocket@gmail.com) |
| 運営責任者 | 請求があった場合には、法令に基づき遅滞なく開示いたします。 |
| 販売価格 | 各商品ページ、申込ページ、または Stripe Checkout の決済ページに表示された金額（消費税込） |
| 商品代金以外の必要料金 | インターネット接続料金、通信料金、端末、OS、Codex CLI、Git など、利用環境の準備や利用に必要な費用はお客様の負担となります。デジタルサービスのため、配送料は発生しません。 |
| 支払方法 | クレジットカード、および決済ページに表示される支払方法 |
| 支払時期 | クレジットカード決済は、お申し込み時にただちに処理されます。継続課金の場合は、契約内容に従い各更新日に決済されます。 |
| 商品またはサービスの提供時期 | 決済完了後、対象機能またはサービスの利用権が確認でき次第、速やかに提供します。 |
| 返品・キャンセル | デジタルサービスの性質上、提供開始後のお客様都合による返品、キャンセル、返金はお受けしていません。 |
| 不良品・サービス不具合への対応 | サービスに不具合がある場合は、上記メールアドレスまでご連絡ください。当方で不具合を確認した場合は、状況に応じて修補、代替手段の案内、または返金を行います。 |
| 継続課金の解約 | 継続課金は、案内されたアカウント管理ページまたはサポートへのご連絡により解約できます。解約後も、支払い済みの利用期間が終了するまでは対象サービスを利用できます。法令上必要な場合または当方が個別に認める場合を除き、日割り返金は行いません。 |
| 申込期間 | 申込ページに期限の表示がある場合は、当該期限までお申し込みいただけます。特段の表示がない場合は、随時お申し込みいただけます。 |
| 販売数量の制限 | 申込ページに数量制限の表示がある場合は、当該表示に従います。特段の表示がない場合、数量制限はありません。 |
| 動作環境 | CodexPocket の利用には、対応する iPhone、Mac、ネットワーク環境、Codex CLI、Git などが必要です。詳しくは [必要なもの](/docs/setup/requirements) と [対応環境](/docs/setup/supported-environments) をご確認ください。 |
