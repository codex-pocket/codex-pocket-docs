このプライバシーポリシーは、CodexPocket の iPhone アプリ（以下「本アプリ」）に適用されます。現行の公開リリースは、ユーザー自身の Mac と iPhone を同一ローカルネットワーク上で接続する companion app として設計されています。

## 要点

- 現行バージョンの本アプリは、開発者のサーバーへユーザーデータを送信して保存する仕組みを実装していません。
- 本アプリは、広告 SDK、トラッキング SDK、第三者分析 SDK を組み込んでいません。
- 現行バージョンの本アプリは、account 作成やログインを要求しません。
- 本アプリで扱う thread、project、接続情報、画像添付は、主に iPhone 上、またはユーザー自身が管理する Mac 上で処理されます。
- 本アプリは、ユーザーデータを販売しません。

## 本アプリが扱う情報

### 1. 開発者が収集するデータ

現行バージョンでは、開発者またはその第三者パートナーがアクセスできる形で、本アプリから個人データを収集する仕組みはありません。

### 2. iPhone 上に保存される情報

本アプリは機能提供のため、次の情報を iPhone に保存することがあります。

- 保存した Host の表示名、接続先、任意メモ
- Bridge 接続用トークン
- アプリ設定（言語、テーマ、フォントサイズ、接続設定など）
- project 一覧や thread 表示のためのキャッシュ
- ユーザーが選択した画像添付の一時データ

Host 情報と bridge token は Keychain に安全に保存されます。Keychain の同期が有効な場合、これらの情報は Apple の Keychain / iCloud Keychain の仕組みにより同期されることがあります。

### 3. ユーザー自身の Mac に直接送られる情報

本アプリは、本来の機能を提供するため、ユーザーが操作した内容をユーザー自身の Mac に直接送信します。たとえば次の情報です。

- thread への入力テキスト
- command、exec、branch 操作に必要な入力
- project や thread の参照に必要な識別情報
- ユーザーが明示的に選択した画像添付

これらのデータは、開発者のサーバーではなく、ユーザー自身が管理する Mac で処理または保持されます。

## 利用する権限

| 権限 | 利用目的 |
| --- | --- |
| ローカルネットワーク | ローカルネットワーク上の Mac bridge を検出して接続するため |
| カメラ | ペアリング QR を読み取るため |
| 写真 | ユーザーが明示的に選択した画像を添付するため。システムの写真 picker を利用し、ライブラリ全体への恒常アクセスは求めません |
| 通知 | task 完了などを iPhone 上で知らせるローカル通知を表示するため。現行バージョンでは、開発者サーバー向けの remote push token 登録は行いません |

本アプリは、連絡先、位置情報、マイク、広告 ID を利用しません。

## 情報の利用目的

本アプリが端末内またはユーザー自身の Mac 上で情報を扱う目的は次のとおりです。

- ユーザー自身の Mac と iPhone を接続するため
- project や thread を再表示し、前回の状態から操作を再開しやすくするため
- bridge token など接続に必要な情報を安全に保持するため
- ユーザーが選択した画像添付や通知を処理するため

## 第三者提供・共有

現行バージョンでは、ユーザーデータを広告配信事業者、データブローカー、外部分析事業者その他の第三者へ共有しません。

また、現行バージョンの本アプリには、第三者の広告 SDK、トラッキング SDK、分析 SDK は含まれていません。

将来、ユーザーデータへアクセスできる第三者サービスを追加する場合は、このページで示す内容以上の保護水準を前提にし、本ポリシーと App Store 上の開示内容を公開前に更新します。

本アプリから外部サイトを開いた場合、そのサイト上でのデータ処理は各サイト運営者のプライバシーポリシーに従います。

## 保持期間

- iPhone 上の設定、キャッシュ、一時データは、ユーザーが本アプリを使用している間、または新しいデータで上書きされるまで端末内に保持されることがあります
- Keychain に保存された Host 情報と bridge token は、ユーザーが該当 Host を削除するまで保持されることがあります
- ユーザー自身の Mac に送られたデータの保持期間は、その Mac 上の設定や保存状態に従います
- 現行バージョンでは、開発者のサーバー側に保持される本アプリ由来のユーザーデータはありません

## プライバシー設定と削除 {#privacy-choices}

ユーザーは次の方法で同意撤回、権限管理、データ削除を行えます。

- `設定` アプリから、本アプリに対するローカルネットワーク、カメラ、写真、通知の権限を変更できます
- 本アプリ内で保存済みの Host を削除すると、対応する Host 情報と bridge token の削除を試みます
- 本アプリを削除すると、アプリコンテナ内の設定やキャッシュは端末から削除されます
- ただし、Keychain または iCloud Keychain に同期された項目は、Apple の Keychain の挙動に従って残る場合があります。そのため、アプリ削除前に Host を削除することを推奨します

現行バージョンでは、開発者が管理するサーバー側に本アプリ由来の個人データを保持していないため、通常、このアプリについて開発者が削除できるサーバー側データはありません。

## 子どもの利用について

本アプリは、13 歳未満の子どもを主な対象として設計したものではありません。

## 本ポリシーの変更

本アプリの機能やデータの取扱いが変わる場合、このページを更新します。将来、アカウント機能、クラウド機能、push 配信、外部サービス連携などを追加する場合は、公開前に本ポリシーと App Store 上のプライバシー開示内容を見直します。
