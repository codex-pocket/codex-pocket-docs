---
title: 初回起動を済ませる
description: 初回起動時の権限と設定の確認方法をまとめます。
---

初回起動では、あとで困らないように最低限の権限と設定だけ確認しておくとスムーズです。

## 権限でよく出るもの

- カメラ
  QR で Mac を追加するときに使います
- ローカルネットワーク
  同じネットワーク上の Mac を見つけて接続するときに使います

QR を使わずに Pairing Code を貼り付けるなら、カメラはあとから許可しても構いません。

## 先に調整しておくと見やすい項目

`設定` で次を変更できます。

- 言語
- テーマ
- フォントサイズ
- フォローアップの既定動作
- KeepAlive
- タイムアウト

ふだん Mac で長いログを読む人は、最初にフォントサイズだけ調整しておくと見やすくなります。

## 初回起動時点では正常なこと

- Host が 0 件
- Project が 0 件
- Thread がまだない

この状態のまま `ホスト` タブへ進み、Mac を取り込んでください。
