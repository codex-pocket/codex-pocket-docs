## 現在の配布状況

- App Store: 現在、公開ページはありません
- TestFlight: 現在、公開リンクはありません
- Mac stable: `0.1.3` を公開中

公開 App Store / TestFlight リンクがない間も、このページで初回画面と最初に確認することを先に把握できます。

CodexPocket の iPhone アプリを入手して起動すると、初回はまだ Mac を取り込んでいないため、空の状態から始まります。

## 初回起動で見る画面

起動直後は、まだ Host がないことを示す案内が表示されます。これは正常です。先に Mac 側の準備が必要なことを知らせています。

## 入手直後に確認すること

- 空の案内と `QR から追加` ボタンが見える
- `Host を開く` から Host 一覧を開ける
- `設定` で言語や文字サイズを変えられる

## まだ Mac を取り込まないでやってよいこと

- アプリの見た目を調整する
- 文字サイズを変える
- 言語を切り替える

ただし、Project や Thread を使い始めるには Host の追加が必要です。
