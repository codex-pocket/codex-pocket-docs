Mac 側の準備が終わったら、次は iPhone 側です。iPhone アプリでは、Mac を `Host` として追加し、Project や Thread を開けるようにします。

入手先と現在の配布状況は、先に [iPhone アプリを入手する](./getting-started) にまとめています。

## iPhone アプリで使う主な画面

- `メイン画面`
  既定 Host、最近使った Project、いま開いている Thread の入口になります
- `サイドバー`
  `プロジェクト` `ホスト` `設定` を開く入口です
- `プロジェクト`
  開きたい workspace を選びます
- `ホスト`
  Mac の追加、接続テスト、再設定を行います
- `設定`
  言語、テーマ、文字サイズ、接続まわりを調整します

## iPhone 側でやること

1. [iPhone アプリを入手する](./getting-started)
2. 必要ならカメラとローカルネットワークを許可する
3. 空の画面の `QR から追加` か、サイドバーの `ホスト` から QR で Mac を取り込む
4. 自動で Project 一覧が入るのを待つ
5. Project と Thread を開く

iPhone 側での主役は `ホスト` と `プロジェクト` です。最初はこの 2 つだけ覚えれば十分です。
