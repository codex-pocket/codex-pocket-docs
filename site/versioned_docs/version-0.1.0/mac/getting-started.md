---
title: Mac アプリをインストールする
description: CodexPocketMac を起動して最初に確認することをまとめます。
---

CodexPocket を iPhone で使うには、Mac 側で `CodexPocketMac` が動いている必要があります。

## まずやること

1. Mac に CodexPocketMac を入れる
2. アプリを起動する
3. `一般 / Bridge / プロジェクト / ペアリング / ログ / 詳細` の各タブが開けることを確認する

起動直後に `CodexPocket Mac` ウィンドウが開き、`一般` の `ランタイム` が見えれば、最初の入口までは到達できています。

## 起動直後に見ておきたい項目

- `Bridge を有効にする`
- `ログイン時に起動`
- 警告表示

特に `Project が 1 件もありません` や `Bridge の前提条件を満たしていません` という表示がある場合は、そのまま先へ進まず次のページを確認してください。

## メニューバーからできること

CodexPocketMac はメニューバーからも操作できます。

- `Bridge を開始`
- `Bridge を停止`
- `ペアリングQRを表示`
- `設定を開く`
- `ログを開く`

Mac を閉じているときでも、メニューバーから Pairing QR をすぐ出せるので便利です。
