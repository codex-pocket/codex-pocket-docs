---
title: Git を使えるようにする
description: ブランチ機能を使う人向けに Git の確認方法をまとめます。
---

Git は必須ではありません。ですが、iPhone から次の操作をしたいなら必要です。

- 現在のブランチ確認
- ブランチ切り替え
- 新しいブランチ作成

## まず確認する

ターミナルで次を実行します。

```bash
git --version
```

バージョンが表示されれば使えます。

## Git がなくても使えること

- Project を開く
- Thread を読む
- Composer から依頼を送る
- Exec を実行する

つまり、「iPhone から続きを読む・送る」だけなら Git はなくても始められます。

## Git が見つからないとき

- macOS の Command Line Tools を入れる
- ふだん使っている Git のパスが Mac アプリから見えるようにする
- Mac アプリを再起動して `詳細` タブの `git` 行を確認する

Git が使えないときは、iPhone 側のブランチ画面で `この workspace は Git 管理されていないため、ブランチ操作は利用できません。` と表示されることがあります。
