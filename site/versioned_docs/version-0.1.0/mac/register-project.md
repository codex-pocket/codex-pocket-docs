Project は、iPhone から開く workspace の単位です。まずは普段使っている workspace を 1 件以上登録してください。

## 登録方法

Mac アプリの `プロジェクト` タブで次の手順を行います。

1. `プロジェクトを追加` を押す
2. `プロジェクト名` を入れる
3. `作業ディレクトリ` を指定する
4. `保存` を押す

`参照…` を使うと Finder からフォルダを選べます。

## どの Project を入れるべきか

おすすめは、ふだん Mac で Codex を使う workspace です。

- いつも開くリポジトリ
- レビューや調査でよく見るディレクトリ
- 外出先から状況確認したい workspace

## Codex App を使っている場合

Mac の Codex App に登録済みの workspace は、CodexPocketMac の `プロジェクト` に自動で追加されることがあります。見当たらない場合だけ、ここで手動追加してください。

## 最初は 1 件あれば十分

Pairing 前に 1 件あると、iPhone で接続した直後から Project 一覧が見えます。あとから増やすこともできます。
