---
title: Project を追加する
description: Codex App の自動反映を使わないときは、iPhone 側から Project を追加します。
---

Project は、iPhone から開く workspace の単位です。Codex App の自動反映で足りないときだけ、iPhone 側から 1 件ずつ追加してください。

## 追加前に Mac で確認すること

- `CodexPocketMac` が起動している
- `Bridge` が `実行中` になっている
- iPhone 側で対象の Host を追加済み

## 追加方法

1. iPhone の `プロジェクト` を開く
2. 右上の `+` から `プロジェクトを追加` を押す
3. Host を選ぶ
4. `プロジェクト名` を入れる
5. `作業ディレクトリ` を指定する
6. `保存` を押す

`Browse on Mac` を使うと、Mac 側のフォルダを見ながら選べます。

## どの Project を入れるべきか

おすすめは、ふだん Mac で Codex を使う workspace です。

- いつも開くリポジトリ
- レビューや調査でよく見るディレクトリ
- 外出先から状況確認したい workspace

## Codex App を使っている場合

Mac の Codex App に登録済みの workspace は、CodexPocketMac の `プロジェクト` に自動で追加されることがあります。見当たらない場合だけ、iPhone 側で手動追加してください。

## 最初は 1 件あれば十分

Pairing 前に 1 件あると、iPhone で接続した直後から Project 一覧が見えます。あとから増やすこともできます。
