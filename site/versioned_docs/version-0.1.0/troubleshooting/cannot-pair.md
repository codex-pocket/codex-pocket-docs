---
title: Pairing できないとき
description: QR や Pairing Code で Mac を追加できないときの確認項目をまとめます。
---

Pairing できないときは、次の順番で確認してください。

## 1. Mac 側の準備を確認する

- CodexPocketMac が起動している
- `ランタイム` が `実行中`
- `ペアリング` 画面で QR が表示できる

ここが整っていないと、iPhone 側だけ直しても進みません。

## 2. iPhone 側の権限を確認する

- QR を使うならカメラが許可されている
- ローカルネットワークが許可されている

権限を拒否していた場合は、設定アプリから許可し直してください。

## 3. ネットワークを確認する

- Mac と iPhone が同じ Wi-Fi にいる
- Mac がスリープしていない
- 別のネットワークやモバイル回線に切り替わっていない

## 4. Pairing 情報を取り直す

- Mac で QR を開き直す
- `トークンを再生成` してから再度読み取る
- QR が難しければ Pairing URL または Pairing Code を使う

## 5. 追加後の初回設定が止まるとき

Host 保存のあとは、Bridge 確認と Project 取り込みが続きます。途中で止まる場合は次を試してください。

- iPhone を前面に戻して待つ
- `ホスト` 画面から `設定を再試行` を押す
- Mac 側で Bridge を再起動する

それでも進まない場合は、Mac の `ログ` を確認してください。
