困ったときは、次の順番で確認すると切り分けやすくなります。

1. Mac 側で Bridge が動いているか
2. iPhone 側で Host が見えているか
3. Project 一覧が取れているか
4. ログに明確なエラーが出ていないか

## 症状から探す

- QR でつながらない
  `Pairing できないとき`
- Host にはつながるが Project が見えない
  `Project が表示されないとき`
- 原因を詳しく見たい
  `ログを確認する`
- 以前は使えたのに、急につながらなくなった
  `再接続と再 Pairing`

## 先に見るとよい場所

- Mac アプリの `一般`
- Mac アプリの `ログ`
- Mac アプリの `詳細`
- iPhone の `ホスト`

CodexPocket は Mac 側に本体があるので、迷ったら先に Mac アプリを見るのが近道です。
