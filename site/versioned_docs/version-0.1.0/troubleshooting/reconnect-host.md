---
title: 再接続と再 Pairing
description: 前は使えていた Host にもう一度つなぐ手順をまとめます。
---

前は使えていたのに急につながらなくなったときは、いきなり再 Pairing せず、次の順番で試してください。

## 1. まず再接続する

- iPhone 側で `更新` する
- `ホスト` で `接続テスト` を行う
- Project を開き直す

## 2. Mac 側の状態を戻す

- CodexPocketMac を開く
- `ランタイム` が `実行中` か確認する
- 必要なら `Bridge を停止` してから `Bridge を開始` する

## 3. それでもだめなら再 Pairing する

次のようなときは再 Pairing が有効です。

- トークンを再生成した
- Mac 側の設定を大きく変えた
- Host 情報を整理したい

再 Pairing の手順は次のとおりです。

1. Mac で `ペアリング` を開く
2. 新しい QR か Pairing Code を出す
3. iPhone で Host を更新または再追加する

## 覚えておきたいこと

iPhone 側のライブ接続が切れても、Mac 側の作業そのものは続いていることがあります。まずは履歴の再同期を試してから、再 Pairing するか判断してください。
